# Application de Prédiction de Prix Immobilier

## Description
Application web interactive pour prédire les prix immobiliers basée sur l'apprentissage supervisé.

## Démonstration
**[Accéder à l'application](https://votre-username-house-price-app-xyz.streamlit.app/)**

## Fonctionnalités
- Exploration interactive des données
- Visualisations dynamiques (Plotly)
- Prédiction de prix avec Random Forest
- Interface accessible (RGAA)
